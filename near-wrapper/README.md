# Near Wrapper 
Implementation of a Near polywrapper.

# Near Plugin
The Near wrapper depends on the [Near Polywrap plugin](https://www.npmjs.com/package/@cidt/near-plugin-js).

# How To Run

## Install Dependencies
`nvm install && nvm use`  
`yarn`

## Build Wrapper
`yarn build`

## Run Test Query Recipe
`yarn test`
